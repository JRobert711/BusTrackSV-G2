# Utils

This directory contains utility functions and helpers.

## Purpose
- JWT token generation/verification
- Validation helpers (using Joi)
- Common helper functions
- Pure functions with no side effects

## Examples
- `jwt.util.js` - Token management
- `validation.util.js` - Input validation schemas
- `logger.util.js` - Logging utilities
- `errors.util.js` - Custom error classes
